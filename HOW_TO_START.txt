================================================================================
  RSCOE PROJECT MANAGEMENT SYSTEM - QUICK START GUIDE
================================================================================

🚀 EASIEST WAY TO START EVERYTHING:

   Simply double-click: START_ALL.bat

   That's it! The script will:
   ✓ Start MongoDB (if not running)
   ✓ Install dependencies (if needed)
   ✓ Start Backend Server (port 5000)
   ✓ Start Frontend Server (port 3000)
   ✓ Open browser automatically

================================================================================
  WHAT HAPPENS WHEN YOU RUN START_ALL.bat:
================================================================================

1. MongoDB Check
   - Checks if MongoDB is already running
   - If not, tries to start it automatically
   - Creates data directory if needed

2. Backend Setup
   - Installs dependencies (first time only)
   - Starts backend server on port 5000
   - Opens in a new window

3. Frontend Setup
   - Installs dependencies (first time only)
   - Starts Vite dev server
   - Opens in a new window

4. Browser Opens
   - Automatically opens http://localhost:3000
   - You're ready to use the application!

================================================================================
  IMPORTANT NOTES:
================================================================================

⚠️  Keep ALL server windows open while using the application!

⚠️  To stop everything:
   - Close the server windows
   - Or press Ctrl+C in each server window

⚠️  If MongoDB is not installed:
   - Download from: https://www.mongodb.com/try/download/community
   - Install MongoDB Community Edition
   - Then run START_ALL.bat again

================================================================================
  TROUBLESHOOTING:
================================================================================

Problem: "MongoDB not found"
Solution: Install MongoDB or start it manually before running the script

Problem: "Port 5000 already in use"
Solution: Close any application using port 5000, then try again

Problem: "Port 3000 already in use"
Solution: Vite will automatically use the next available port

Problem: "npm install failed"
Solution: Check your internet connection and try again

Problem: Browser didn't open
Solution: Manually visit http://localhost:3000
